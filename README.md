Sudoku Solver 
==========

Simple Sudoku solver built in C.

How to Use
==========

Declare the puzzle as a 9x9 2D array and feed it to the solve function. Replace the unknown cells with 0.

```c
int sudoku[9][9]={
                        {3, 0, 6, 0, 0, 0, 5, 0, 0},
                        {0, 7, 0, 0, 6, 0, 0, 8, 9},
                        {0, 5, 9, 0, 0, 2, 4, 0, 6},
                        {9, 0, 0, 0, 5, 1, 0, 0, 0},
                        {0, 0, 5, 0, 0, 0, 8, 0, 0},
                        {6, 0, 0, 2, 4, 0, 0, 0, 3},
                        {5, 9, 0, 0, 8, 6, 1, 2, 7},
                        {1, 6, 0, 0, 0, 0, 0, 0, 8},
                        {0, 8, 0, 0, 0, 0, 0, 0, 0}
                    };
```
